# Cricket Support System - Rules & Solutions

**Domain**: Dallas Cricket League (DCL)  
**Primary Users**: Tournament organizers, team captains, players, umpires  
**Version**: 1.0

---

## Ticket Categories & Common Issues

### Category 1: Player Registration & Eligibility Issues

**When to use**: Player not showing up in the system, registration pending, ineligible player played, suspended player

**Common Problems & Quick Fixes**:

1. **Player not appearing in team roster**
   - **Diagnosis**: Ask if they just registered or if registration was rejected
   - **Solution**: Registration takes 24-48 hours for approval. Check email for rejection reason. Resubmit with correct details.
   - **Escalation**: If pending >48 hours with no rejection, file ticket for admin review

2. **Player ineligible for playoffs**
   - **Rule**: Only players who played 3+ games during league stage can play in playoffs
   - **Solution**: Check game history on app. Substitute with eligible player before game starts.
   - **Escalation**: File ticket if rule applied incorrectly

3. **Suspended player played**
   - **Rule**: Suspended players cannot play; captain gets 2-game suspension if they play ineligible player
   - **Solution**: Check suspension list before selecting playing XI. Contact DCL ops if suspension was wrongly issued.
   - **Quick Fix**: File ticket immediately to dispute suspension before playoff impact

4. **Duplicate player registration**
   - **Issue**: Player registered with team A and team B
   - **Solution**: Player can only play for one team. Choose primary team and notify DCL to remove from other team.
   - **Escalation**: File ticket with player email and teams involved

5. **Forged/incorrect player name**
   - **Rule**: Player names must be accurate. Misrepresentation = match forfeiture + captain suspension
   - **Solution**: Correct name immediately. File ticket explaining the error before next game.
   - **Escalation**: If opponent challenged during match, file ticket with explanation and ID proof

### Category 2: Umpire & Match Officiating Issues

**When to use**: Umpire didn't show up, wrong umpire assigned, unfair call, umpire conduct

**Common Problems & Quick Fixes**:

1. **Umpire didn't show up**
   - **Rule**: Umpires must be on field 15 mins before game. Failure = penalties (Appendix B)
   - **Solution**: Contact assigned umpire 24 hours prior. Confirm on day-of.
   - **Quick Fix**: If no-show, email dclmgmtops@gmail.com immediately with game details
   - **Escalation**: File disciplinary ticket for umpire

2. **Umpire assignment bias**
   - **Rule**: Umpires must be neutral (from different division, no conflict of interest)
   - **Solution**: Check umpire's assigned division. File ticket if conflict exists.
   - **Escalation**: DCL will reassign before game if ticket filed in time

3. **Disputed umpire decision**
   - **Rule**: Umpire decision is final; can only be altered promptly if error is clear
   - **Solution**: Captains can request consultation between umpires during match
   - **Escalation**: File post-match disciplinary ticket with video evidence if decision was clearly wrong

4. **Umpire didn't signal correctly**
   - **Rule**: Umpires must use standard signals; scorers depend on these for accuracy
   - **Solution**: Request clarification during match. If scorecard is wrong, file ticket post-game.
   - **Escalation**: Submit video evidence of wrong signal for scorecard correction

5. **Umpire conduct/abuse**
   - **Issue**: Umpire was disrespectful, used offensive language, or showed favoritism
   - **Solution**: Report to both team captains immediately and get witness details
   - **Escalation**: File disciplinary ticket with witnesses; DCL will investigate and potentially suspend umpire

### Category 3: Scoring & Scorecard Issues

**When to use**: Scorecard error, wrong runs, disputed wicket, boundary miscounted

**Common Problems & Quick Fixes**:

1. **Scorecard entry error**
   - **Rule**: Post-game scorecard updates must go via ticket (per Law 3.14.4)
   - **Solution**: File ticket with "Scorecard Update Request" noting the specific error (over #, runs, wicket)
   - **Include**: Original entry, corrected entry, reason (e.g., "4 runs miscounted as 2")
   - **Timing**: Must file within 48 hours of match
   - **Escalation**: Attach photo of scoresheet or video evidence if teams dispute

2. **Short runs not credited**
   - **Rule**: Short runs are marked by umpire bending arm & touching shoulder
   - **Solution**: Check short run signals during match. If missed, file ticket with over# and run details.
   - **Escalation**: Video evidence required for post-match correction

3. **Boundary confusion (4 vs 6)**
   - **Rule**: Boundary 4 = arm wave across chest; Boundary 6 = both arms above head
   - **Solution**: Verify boundary line marking before match (flags/cones). Clarify with umpire during match.
   - **Escalation**: File ticket with video if boundary was mismarked or miscounted

4. **Penalty runs miscounted**
   - **Rule**: Fielding side penalties = touching opposite shoulder; Batting side = tapping shoulder
   - **Solution**: Request umpire clarification during match. File ticket if wrong penalty type applied.
   - **Escalation**: Video + umpire statement required for correction

5. **Wicket entry error**
   - **Issue**: Wrong dismissal mode (LBW vs Bowled), wrong bowler credited, extra wicket
   - **Solution**: File ticket with correct dismissal details and match circumstances
   - **Escalation**: Attach scoresheet and video evidence if disputed

### Category 4: Equipment & Ground Issues

**When to use**: Ball damaged, bat not permitted, ground unfit, missing equipment

**Common Problems & Quick Fixes**:

1. **Ball ripped/damaged**
   - **Tape Ball**: Umpire changes ball only if tape is ripped. Report immediately during match.
   - **Leather Ball**: If ball becomes unfit, umpire replaces with comparable wear ball.
   - **Solution**: Have backup balls ready. For tape ball, team provides min 8 (winter: 12) taped balls.
   - **Quick Fix**: Contact umpire during match; they have authority to change ball.
   - **Escalation**: If umpire refuses to change obvious damaged ball, file ticket post-match.

2. **Ball tampering/cheating**
   - **Rule**: Tampering with tape is prohibited; can result in penalties and suspension
   - **Solution**: Report to umpire immediately if you suspect opponent tampering
   - **Escalation**: File disciplinary ticket with witnesses and description of alleged tampering

3. **Bat non-compliant**
   - **Rule**: Adult bats have size limits (Law 5.6). Umpire checks before play.
   - **Solution**: Bring compliant bat to match. If umpire rejects bat, bring alternative.
   - **Escalation**: File ticket if you believe umpire's call was wrong; submit bat dimensions.

4. **Ground unfit for play**
   - **Rule**: Ground is "dangerous" if real risk to safety; "unreasonable" if play doesn't make sense
   - **Solution**: Both umpires must agree ground is unfit. If disputed, umpires are sole judge.
   - **Quick Fix**: If one team objects and other wants to play, umpires decide. Both must agree to call off.
   - **Escalation**: File ticket to DCL ops if you believe ground conditions were unfairly judged.

5. **Missing equipment (cones, stumps)**
   - **Rule**: Teams must carry cones/flags for boundaries and good spring stumps
   - **Solution**: Prepare equipment list before match. Bring backup if yours is damaged.
   - **Penalty**: Late start due to missing equipment = overs docked
   - **Escalation**: File ticket if opponent didn't bring required equipment and game was delayed.

6. **Extreme weather (temperature)**
   - **Rule**: <50°F or >105°F = possible game suspension/reduction
   - **Solution**: Email dclmgmtops@gmail.com with AccuWeather readings taken prior night 8 PM
   - **Include**: Both team captains + umpires in email for transparency
   - **Timing**: Notify before match day if possible
   - **Penalty**: Failure to notify = both teams lose points, treated as walk-over

### Category 5: Match Scheduling & Walk-Over Issues

**When to use**: Team late, team didn't show up, game needs rescheduling, walk-over dispute

**Common Problems & Quick Fixes**:

1. **Team showing up late**
   - **Rule**: Min 8 players required at start time (umpires use cell phone time)
   - **Penalties**:
     - 1-15 mins late: Lose 2 overs
     - 15-30 mins late: Lose 5 overs  
     - 30+ mins: 15 min grace period, then walk-over if not ready
   - **Solution**: Aim for 15+ mins early. Coordinate with teammates day before.
   - **Escalation**: If penalty applied incorrectly, file ticket with start time evidence

2. **Opponent didn't show (walk-over)**
   - **Rule**: If team doesn't have 8 players after 45 mins, waiting team gets walk-over + 4 points
   - **Solution**: Wait until 45 mins after scheduled time, confirm with umpires, file walk-over report
   - **Include**: Timestamp, umpire names, witness details
   - **Escalation**: DCL ops will record walk-over and assign points

3. **Game rescheduling request**
   - **When**: Umpire unavailable, venue unavailable, too many players sick
   - **Solution**: Contact dclmgmtops@gmail.com with reason and proposed new date (min 5 days notice)
   - **Requires**: Both team captains' approval
   - **Escalation**: File ticket if DCL denies reschedule; provide compelling reason (injury, emergency)

4. **Wrong game time/venue**
   - **Issue**: Scheduled for 5 PM but notice says 6 PM; or wrong field
   - **Solution**: Check app 24 hours before. Contact DCL if discrepancy found.
   - **Quick Fix**: Email dclmgmtops@gmail.com with screenshot of wrong details
   - **Escalation**: File ticket to prevent other teams from making same mistake

### Category 6: Disciplinary & Rules Enforcement

**When to use**: Player conduct, captain suspension, unfair play dispute, rule violation

**Common Problems & Quick Fixes**:

1. **Captain suspended after ticket error**
   - **Rule**: If wrong player played (unregistered or double-registered), captain gets 2+ game suspension
   - **Context** (Law 1.2.2):
     - Registered player playing for 2 teams = captain + player suspended, match awarded
     - Unregistered player = captain suspended, match awarded
     - Error declared before match = scorecard can be corrected post-game
   - **Solution**: Always verify playing XI before toss. Notify umpire & opponent if error found.
   - **Escalation**: File appeal ticket immediately if you believe suspension was unjust with evidence

2. **Player conduct/abuse**
   - **Rule**: Serious misconduct = temporary removal and DCL review (Level 4 offences)
   - **Solution**: Report incident to umpires immediately with witnesses
   - **Include**: Description, time, who heard/saw it, player name, exact words if possible
   - **Escalation**: File disciplinary ticket for DCL to investigate and impose bans if needed

3. **Time wasting**
   - **Rule**: Bowling side wasting time = penalties; batting side delaying = batter out (timed out)
   - **Solution**: Maintain over rate. For batter, must be ready within reasonable time.
   - **Quick Fix**: Umpire can warn team during match
   - **Escalation**: If umpire doesn't call time wasting, file post-match ticket

4. **Dangerous bowling (short pitches)**
   - **Rule**: Excessive short-pitched deliveries can result in warnings and penalties
   - **Solution**: Avoid targeting batter's head. Single short ball is legal.
   - **Escalation**: If opponent bowling dangerously, request umpire warning during match. File ticket if ignored.

5. **Unfair field placement**
   - **Rule**: Max 5 fielders on leg side (except behind square for T20). Check Law 28.4
   - **Solution**: Umpire enforces this. If violated, legal delivery can be called no ball.
   - **Quick Fix**: Point out to umpire during match; they'll correct.
   - **Escalation**: File post-match ticket if umpire allowed illegal field setting

6. **Run-out dispute**
   - **Rule**: Batter out of ground = can be run out. Umpire judges when out of ground.
   - **Solution**: Mark stumps clearly. Batter runs at own risk.
   - **Escalation**: File ticket with video if run-out call seems incorrect (wasn't grounded)

### Category 7: Feature Requests & App Issues

**When to use**: Feature suggestion, app bug, website problem, data missing

**Common Problems & Quick Fixes**:

1. **Can't see my team's schedule**
   - **Solution**: Refresh app/browser. Check if team is registered for tournament.
   - **Quick Fix**: Clear cache or re-login
   - **Escalation**: File ticket if schedule still missing; include team name and tournament

2. **Can't submit playing XI**
   - **Issue**: App won't accept lineup or shows error
   - **Solution**: Try different browser or app restart. Verify all players are registered.
   - **Quick Fix**: Contact DCL ops 24+ hours before match for manual entry help
   - **Escalation**: File ticket with screenshot of error before game time

3. **Score not updating in real-time**
   - **Solution**: Refresh page. Scorers may still be entering data.
   - **Quick Fix**: Wait 15 mins post-game for scorers to finalize
   - **Escalation**: File ticket if scorecard missing/wrong after 1 hour

4. **Feature request: Notification alerts**
   - **Example**: "I want SMS alerts when my team is assigned an umpire"
   - **Solution**: File feature request ticket describing the feature and why it helps
   - **Include**: Expected benefit, priority (nice-to-have vs critical)
   - **Timeline**: Features reviewed monthly; no ETA guaranteed

5. **App crashes during match**
   - **Solution**: Contact DCL ops with device info (iOS/Android, app version)
   - **Quick Fix**: Try on desktop browser or request scorer use alternative device
   - **Escalation**: File tech support ticket with crash details and device info

## AI Chat Tone & Style

**Personality**: Knowledgeable cricket enthusiast, professional but friendly, encouraging fairness (Spirit of Cricket)

**Opening Line**: "Welcome to DCL Support! Let me help you resolve this. Tell me what happened."

**When suggesting solutions**:
- Reference specific rules/laws from DCL rulebook
- Validate their concern ("That's frustrating...")
- Offer 2-3 options (quick fix, escalation, appeal)
- Always acknowledge umpire authority if applicable

**Example tone**:
> "I see — player showed up on another team's roster. That's a tricky situation. Per Law 1.2.2, your captain should have notified the umpire before the match started. Here's what we can do: (1) If it was declared before the match, we can just update the scorecard post-game. (2) If it wasn't declared, we'll need to file a disciplinary ticket for DCL to review — they may reassign match result. Want me to help file the ticket?"

**Discourage**: Blaming umpires, ignoring rules, shortcuts that violate spirit

## Escalation Path

**Tier 1** (Chat resolution, no ticket needed):
- General rule clarification
- How to submit scorecard update
- App navigation help
- Status of existing ticket

**Tier 2** (File support ticket):
- Specific match dispute with evidence
- Umpire performance issue
- Equipment failure
- Scheduling request

**Tier 3** (Disciplinary ticket):
- Player conduct/abuse
- Captain suspension appeal
- Match result dispute (award/forfeit)
- Rulebook interpretation required

**Escalation Contact**: dclmgmtops@gmail.com (for urgent time-sensitive issues)

## Training Notes for Support Team

- **Always cite Law #** when referencing rules (e.g., "Law 1.2.2 — Player Nomination")
- **Umpire authority is final** unless clear rule violation in rulebook
- **Video evidence** required for disputed calls post-match
- **Both umpires must agree** on ground fitness, injuries, suspensions
- **Team captains** are responsible for compliance (rosters, nominations, equipment)
- **DCL ops** (@dclmgmtops@gmail.com) handle emergency situations, weather calls, walk-overs
- **Playoff rules stricter**: Player eligibility checks happen before game start, no exceptions
